# LogBookReaderWPF

Приложение "читает" файл журнала регистрации 1С формата SQLite.  
  
На текущий момент реализованы следующие фильтры: 
- Период
- Приложение
- Компьютер
- Событие
- Пользователи (+ фильтр списка пользователей)
- Только записи с комментариями
